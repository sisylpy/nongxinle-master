# 农信乐微信小程序前端开发指南

## 📱 项目简介

**农信乐**是一个社区配送电商平台，主要服务于社区饭馆和顾客之间的配送业务。

### 核心业务场景

1. **顾客端**：顾客可以在小程序中选择地址，查找附近的饭馆，下单购买商品
2. **饭馆端**：饭馆可以管理商品、处理订单、管理配送
3. **集团管理**：多个饭馆可以归属于一个集团，统一管理

---

## 🏗️ 系统架构

### 实体关系

```
集团（ECommerce）
  └── 饭馆（Community）
      ├── 商品（Goods）
      ├── 订单（Orders）
      └── 顾客（Customer）
          └── 地址（Address）
```

### 核心概念

- **集团（ECommerce）**：多个饭馆的集合，统一管理
- **饭馆（Community）**：提供商品和配送服务的实体店
- **顾客（Customer）**：购买商品的用户
- **地址（Address）**：顾客的收货地址，包含坐标信息

## 📍 坐标系统说明

### 坐标的重要性

坐标是整个系统的核心，用于：
- ✅ 顾客查找附近可配送的饭馆
- ✅ 计算配送距离和费用
- ✅ 确定配送范围

### 坐标格式

- **经度（lng）**：-180 到 180
- **纬度（lat）**：-90 到 90
- **格式**：字符串类型，例如 `"117.07822"`、`"39.98246"`

### 获取坐标的方法

**微信小程序获取位置**：

```javascript
// 方式1：获取当前位置
wx.getLocation({
  type: 'gcj02', // 返回可以用于wx.openLocation的经纬度
  success: (res) => {
    const latitude = res.latitude.toString();   // 纬度
    const longitude = res.longitude.toString(); // 经度
    console.log('纬度：', latitude);
    console.log('经度：', longitude);
  }
});

// 方式2：选择位置（推荐）
wx.chooseLocation({
  success: (res) => {
    const latitude = res.latitude.toString();
    const longitude = res.longitude.toString();
    const address = res.address; // 地址名称
    console.log('位置：', res);
  }
});
```

**注意事项**：
- 需要在 `app.json` 中配置位置权限
- 坐标需要转换为字符串格式传递给后端
- 建议使用 `wx.chooseLocation` 让用户手动选择位置，更准确

## 🔌 核心接口说明

### 1. 顾客地址相关接口

#### 1.1 保存顾客地址

**接口**：`POST /api/nxcustomeruseraddress/userSaveAddress`

**请求参数**：
```json
{
  "nxCuaCustomerUserId": 123,              // 用户ID
  "nxCuaUserName": "张三",                  // 收货人姓名
  "nxCuaUserPhone": "13910825707",         // 收货人电话
  "nxCuaAddressDetail": "详细地址",        // 详细地址
  "nxCuaAddressBuildingName": "建筑物名称", // 建筑物名称
  "nxCuaLat": "39.98246",                  // 纬度（字符串）
  "nxCuaLng": "117.07822",                 // 经度（字符串）
  "nxCuaSysProvinceId": 1,                 // 省份ID
  "nxCuaSysCityId": 2,                      // 城市ID
  "nxCuaSysDistrictId": 3,                 // 区县ID
  "nxCuaSysBusinessAreaId": 4              // 商圈ID
}
```

**响应示例**：
```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "nxCuUserId": 123,
    "mainAddress": {
      "nxCustomerUserAddressId": 456,
      "nxCuaLat": "39.98246",
      "nxCuaLng": "117.07822"
    }
  }
}
```

**前端实现示例**：
```javascript
// 选择位置并保存地址
chooseAndSaveAddress() {
  wx.chooseLocation({
    success: (res) => {
      const addressData = {
        nxCuaCustomerUserId: this.data.userId,
        nxCuaUserName: this.data.userName,
        nxCuaUserPhone: this.data.userPhone,
        nxCuaAddressDetail: res.address + res.name,
        nxCuaAddressBuildingName: res.name,
        nxCuaLat: res.latitude.toString(),
        nxCuaLng: res.longitude.toString(),
        // ... 其他字段
      };
      
      wx.request({
        url: 'https://your-domain.com/api/nxcustomeruseraddress/userSaveAddress',
        method: 'POST',
        data: addressData,
        success: (result) => {
          if (result.data.code === 0) {
            wx.showToast({ title: '地址保存成功' });
            // 刷新地址列表
            this.loadAddressList();
          }
        }
      });
    }
  });
}
```

#### 1.3 根据地址查询附近饭馆 ⭐核心接口

**接口**：`GET /api/nxcustomeruseraddress/getCommityByAddressId/{addressId}`

**功能**：根据地址ID，查询该地址100公里范围内的所有饭馆

**响应示例**：
```json
{
  "code": 0,
  "data": [
    {
      "nxCommunityId": 1,
      "nxCommunityName": "测试饭馆",
      "nxCommunityLat": "39.98246",
      "nxCommunityLng": "117.07822",
      "location": "POINT(117.07822 39.98246)",  // 空间坐标文本
      "distance": 1234.56,                       // 距离（米）
      "nxCommunityDeliveryAddress": "配送地址",
      "nxCommunityOpenTime": "06:00",
      "nxCommunityCloseTime": "22:00"
    }
  ]
}
```

**前端实现示例**：
```javascript
// 根据地址查询附近饭馆
getNearbyRestaurants(addressId) {
  wx.request({
    url: `https://your-domain.com/api/nxcustomeruseraddress/getCommityByAddressId/${addressId}`,
    method: 'GET',
    success: (res) => {
      if (res.data.code === 0) {
        const restaurants = res.data.data;
        // 转换距离单位（米转公里）
        restaurants.forEach(item => {
          item.distanceKm = (item.distance / 1000).toFixed(2);
        });
        // 按距离排序
        restaurants.sort((a, b) => a.distance - b.distance);
        this.setData({ restaurants });
      }
    }
  });
}
```

**使用场景**：
- 用户选择地址后，自动查询附近可配送的饭馆
- 显示饭馆列表，按距离排序
- 显示每个饭馆的距离信息

### 2. 饭馆相关接口

#### 2.1 饭馆注册

**接口**：`POST /api/nxcommunity/comAndUserSave`

**请求参数**：
```json
{
  "nxCommunityName": "饭馆名称",
  "nxCommunityLat": "39.98246",           // 纬度（必填）
  "nxCommunityLng": "117.07822",          // 经度（必填）
  "nxCommunityCommerceId": 1,              // 所属集团ID（必填）
  "nxCommunityDeliveryAddress": "配送地址",
  "nxCommunityType": 1,
  "nxCommunitySysProvinceId": "1",
  "nxCommunitySysCityId": "2",
  "nxCommunitySysDistrictId": "3",
  "nxCommunitySysBusinessAreaId": "4",
  "nxCommunityUserEntity": {
    "nxCouCode": "微信登录code",          // 必填，通过wx.login()获取
    "nxCouWxNickName": "管理员昵称",
    "nxCouWxAvartraUrl": "头像URL",
    "nxCouWxPhone": "手机号"
  }
}
```

**前端实现示例**：
```javascript
// 饭馆注册
registerRestaurant(restaurantInfo) {
  // 1. 获取微信登录code
  wx.login({
    success: (loginRes) => {
      const code = loginRes.code;
      
      // 2. 获取位置信息
      wx.chooseLocation({
        success: (locationRes) => {
          const registerData = {
            nxCommunityName: restaurantInfo.name,
            nxCommunityLat: locationRes.latitude.toString(),
            nxCommunityLng: locationRes.longitude.toString(),
            nxCommunityCommerceId: restaurantInfo.commerceId,
            nxCommunityDeliveryAddress: locationRes.address,
            nxCommunityUserEntity: {
              nxCouCode: code,
              nxCouWxNickName: restaurantInfo.nickName,
              nxCouWxAvartraUrl: restaurantInfo.avatarUrl,
              nxCouWxPhone: restaurantInfo.phone
            }
          };
          
          // 3. 调用注册接口
          wx.request({
            url: 'https://your-domain.com/api/nxcommunity/comAndUserSave',
            method: 'POST',
            data: registerData,
            success: (res) => {
              if (res.data.code === 0) {
                wx.showToast({ title: '注册成功' });
                // 保存用户信息到本地
                wx.setStorageSync('userInfo', res.data.data);
              } else {
                wx.showToast({ title: res.data.msg });
              }
            }
          });
        }
      });
    }
  });
}
```

## 🎯 常见业务场景实现

### 场景1：顾客选择地址后查找附近饭馆

```javascript
Page({
  data: {
    addressList: [],
    selectedAddress: null,
    restaurants: []
  },
  
  onLoad() {
    this.loadAddressList();
  },
  
  // 加载地址列表
  loadAddressList() {
    const userId = wx.getStorageSync('userId');
    wx.request({
      url: `https://your-domain.com/api/nxcustomeruseraddress/userGetAddress/${userId}`,
      success: (res) => {
        if (res.data.code === 0) {
          this.setData({ addressList: res.data.data });
        }
      }
    });
  },
  
  // 选择地址
  selectAddress(e) {
    const addressId = e.currentTarget.dataset.id;
    this.setData({ selectedAddress: addressId });
    this.getNearbyRestaurants(addressId);
  },
  
  // 查询附近饭馆
  getNearbyRestaurants(addressId) {
    wx.showLoading({ title: '查找中...' });
    wx.request({
      url: `https://your-domain.com/api/nxcustomeruseraddress/getCommityByAddressId/${addressId}`,
      success: (res) => {
        wx.hideLoading();
        if (res.data.code === 0) {
          const restaurants = res.data.data.map(item => ({
            ...item,
            distanceKm: (item.distance / 1000).toFixed(2) + 'km'
          }));
          restaurants.sort((a, b) => a.distance - b.distance);
          this.setData({ restaurants });
        }
      }
    });
  }
});
```

### 场景3：饭馆注册

```javascript
// 饭馆注册页面
registerRestaurant() {
  const that = this;
  
  // ⚠️ 重要：必须先获取微信登录code
  wx.login({
    success: (loginRes) => {
      if (!loginRes.code) {
        wx.showToast({ title: '获取登录信息失败' });
        return;
      }
      
      const code = loginRes.code;
      
      // 选择位置
      wx.chooseLocation({
        success: (locationRes) => {
          // ⚠️ 注意：不要传 nxCommunityLocation 字段
          const registerData = {
            nxCommunityName: that.data.restaurantName,
            nxCommunityLat: locationRes.latitude.toString(),  // 字符串格式
            nxCommunityLng: locationRes.longitude.toString(), // 字符串格式
            // ❌ 不要传 nxCommunityLocation
            nxCommunityCommerceId: that.data.commerceId,
            nxCommunityDeliveryAddress: locationRes.address,
            nxCommunityUserEntity: {
              nxCouCode: code,  // ✅ 必填：微信登录code
              nxCouWxNickName: '微信用户'
            }
          };
          
          wx.request({
            url: 'https://your-domain.com/api/nxcommunity/comAndUserSave',
            method: 'POST',
            header: {
              'content-type': 'application/json'
            },
            data: registerData,
            success: (res) => {
              if (res.data.code === 0) {
                wx.showToast({ title: '注册成功' });
                wx.setStorageSync('userInfo', res.data.data);
                // 跳转到饭馆管理页面
                wx.reLaunch({
                  url: '/pages/restaurant/index'
                });
              } else {
                wx.showToast({ title: res.data.msg || '注册失败' });
              }
            },
            fail: (err) => {
              console.error('注册失败：', err);
              wx.showToast({ title: '网络错误' });
            }
          });
        },
        fail: (err) => {
          wx.showToast({ title: '请选择位置' });
        }
      });
    },
    fail: (err) => {
      wx.showToast({ title: '登录失败' });
    }
  });
}
```

**⚠️ 常见错误**：
1. ❌ 缺少 `nxCouCode` 字段 → 会导致500错误
2. ❌ 传递 `nxCommunityLocation` 字段 → 不需要，后端会自动构建
3. ❌ 坐标不是字符串格式 → 必须使用 `.toString()`

## ⚠️ 注意事项

### 1. 坐标相关

- ✅ **坐标必须是字符串格式**：`latitude.toString()`、`longitude.toString()`
- ✅ **经度在前，纬度在后**：后端使用 `POINT(lng lat)` 格式
- ✅ **坐标必填**：注册饭馆和添加地址时，坐标是必填项
- ✅ **位置权限**：需要在 `app.json` 中配置位置权限

### 2. 微信小程序配置

**app.json 配置**：
```json
{
  "permission": {
    "scope.userLocation": {
      "desc": "你的位置信息将用于查找附近的饭馆"
    }
  },
  "requiredPrivateInfos": ["getLocation", "chooseLocation"]
}
```

### 3. 接口调用

- ✅ **统一使用 HTTPS**：所有接口调用必须使用 HTTPS
- ✅ **错误处理**：需要处理网络错误和业务错误
- ✅ **加载提示**：长时间操作需要显示加载提示
- ✅ **用户反馈**：操作成功或失败需要给用户明确反馈

### 4. 距离显示

- 后端返回的距离单位是**米**
- 前端显示时建议转换为**公里**：`(distance / 1000).toFixed(2) + 'km'`
- 小于1公里时显示米：`distance < 1000 ? distance + 'm' : (distance / 1000).toFixed(2) + 'km'`
