# 前端开发快速参考

## 🚀 快速开始

### 1. 获取位置坐标

```javascript
wx.chooseLocation({
  success: (res) => {
    const lat = res.latitude.toString();  // 纬度（字符串）
    const lng = res.longitude.toString(); // 经度（字符串）
  }
});
```

### 2. 保存地址

```javascript
wx.request({
  url: 'https://your-domain.com/api/nxcustomeruseraddress/userSaveAddress',
  method: 'POST',
  data: {
    nxCuaCustomerUserId: userId,
    nxCuaLat: lat,      // 字符串格式
    nxCuaLng: lng,      // 字符串格式
    nxCuaUserName: '姓名',
    nxCuaUserPhone: '电话',
    nxCuaAddressDetail: '详细地址'
  }
});
```

### 3. 查询附近饭馆

```javascript
wx.request({
  url: `https://your-domain.com/api/nxcustomeruseraddress/getCommityByAddressId/${addressId}`,
  success: (res) => {
    const restaurants = res.data.data;
    // distance 单位是米，转换为公里显示
    restaurants.forEach(item => {
      item.distanceKm = item.distance < 1000 
        ? item.distance + 'm' 
        : (item.distance / 1000).toFixed(2) + 'km';
    });
  }
});
```

## 📋 接口清单

| 功能 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 保存地址 | POST | `/api/nxcustomeruseraddress/userSaveAddress` | 保存顾客地址 |
| 获取地址列表 | GET | `/api/nxcustomeruseraddress/userGetAddress/{userId}` | 获取用户所有地址 |
| 查询附近饭馆 | GET | `/api/nxcustomeruseraddress/getCommityByAddressId/{id}` | 根据地址查询100km内饭馆 |
| 删除地址 | DELETE | `/api/nxcustomeruseraddress/deleteUserAddress/{id}` | 删除地址 |
| 编辑地址 | POST | `/api/nxcustomeruseraddress/userEditAddress` | 更新地址信息 |
| 饭馆注册 | POST | `/api/nxcommunity/comAndUserSave` | 注册新饭馆 |
| 查询饭馆信息 | GET | `/api/nxcommunity/getCommunityInfo/{id}` | 获取饭馆详情 |

## ⚡ 常用代码片段

### 距离格式化

```javascript
formatDistance(distance) {
  if (distance < 1000) {
    return distance + 'm';
  } else {
    return (distance / 1000).toFixed(2) + 'km';
  }
}
```

### 获取微信登录code

```javascript
wx.login({
  success: (res) => {
    const code = res.code; // 传给后端
  }
});
```

```javascript
wx.chooseLocation({

```

## ⚠️ 重要提醒

1. **坐标格式**：必须是字符串，使用 `.toString()` 转换
2. **坐标顺序**：后端使用 `POINT(经度 纬度)` 格式
3. **距离单位**：后端返回的是米，前端需要转换显示
4. **位置权限**：需要在 app.json 中配置权限

## 📱 app.json 配置示例

```json
{
  "permission": {
    "scope.userLocation": {
      "desc": "你的位置信息将用于查找附近的饭馆"
    }
  },
  "requiredPrivateInfos": ["getLocation", "chooseLocation"]
}
```
