# 协作伙伴与商品可见性接口说明

> 基础路径：`/api/nxdistributernxdistributer`  
> 请求方式：POST（除特别说明外）  
> Content-Type：`application/json`（Body）或 `application/x-www-form-urlencoded`（表单）

---

## 一、业务说明

### 1. 协作关系

- 两个配送商建立协作后，可互相采购/供货
- 建立时需选择**邀请类型**，决定默认商品可见性

### 2. 商品可见性（类似微信朋友圈）

| 设置项 | 含义 | 效果 |
|--------|------|------|
| **不看他的商品** | 我不看对方的商品 | 我查商品时看不到该伙伴的商品 |
| **不给他看我的商品** | 对方看不到我的商品 | 对方查商品时看不到我的商品 |

双方可独立设置，互不影响。

---

## 二、接口列表

### 1. 协作伙伴列表（含账单、可见性状态）

**路径：** `POST /nxDisGetAllOfferNxDis`

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| nxDisId | Integer | 是 | 当前配送商 id |
| userId | Integer | 否 | 用户 id |
| startDate | String | 否 | 统计开始日期，如 `2026-02-01` |
| stopDate | String | 否 | 统计结束日期，如 `2026-02-28` |

**返回示例：**

```json
{
  "code": 0,
  "msg": "ok",
  "data": [
    {
      "nxDistributerId": 166,
      "nxDistributerName": "XX批发商",
      "nxDistributerShowName": "XX批发",
      "nxDistributerPhone": "13800138000",
      "itemData": {
        "blockedByMe": false,
        "blockedMe": false,
        "billCount": 5,
        "billTotal": "1234.5",
        "unPayCount": 2,
        "unPayTotal": "500.0",
        "havePayCount": 3,
        "havePayTotal": "734.5",
        "unConfirmCount": 1,
        "unConfirmTotal": "200.0"
      }
    }
  ]
}
```

**itemData 字段说明：**

| 字段 | 类型 | 说明 |
|------|------|------|
| blockedByMe | boolean | 我不看他的商品 |
| blockedMe | boolean | 他不看我的商品 |
| billCount | int | 账单总数 |
| billTotal | string | 账单总金额 |
| unPayCount | int | 未结账数量 |
| unPayTotal | string | 未结账金额 |
| havePayCount | int | 已结账数量 |
| havePayTotal | string | 已结账金额 |
| unConfirmCount | int | 待确认数量 |
| unConfirmTotal | string | 待确认金额 |

### 2. 建立协作关系

**路径：** `POST /saveBusiness`

**请求体（JSON）：**

```json
{
  "nxDistributerId1": 100,
  "nxDistributerId2": 200,
  "inviteType": 1,
  "inviterNxDistributerId": 100
}
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| nxDistributerId1 | Integer | 是 | 配送商 1 id |
| nxDistributerId2 | Integer | 是 | 配送商 2 id |
| inviteType | Integer | 否 | 1=申请成为供货商，2=申请采购我的商品；不传则无默认 block |
| inviterNxDistributerId | Integer | inviteType=1 时必填 | 发起邀请的配送商 id，需为 id1 或 id2 之一 |

**邀请类型说明：**

| inviteType | 含义 | 默认可见性 |
|------------|------|------------|
| 1 | 申请成为供货商 | 申请方能看到同意方商品；申请方不给同意方看自己的商品；同意方默认不看申请方商品 |
| 2 | 申请采购我的商品 | 同意方不给申请方看商品；申请方不看同意方的商品 |

---

### 3. 解除协作关系

**路径：** `POST /removePartner`

**参数（表单）：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| myDisId | Integer | 是 | 当前配送商 id |
| partnerDisId | Integer | 是 | 要解除的协作伙伴 id |

---

### 4. 不看他的商品（拉黑）

**路径：** `POST /blockPartner`

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| blockerDisId | Integer | 是 | 当前配送商 id（谁不看） |
| blockedDisId | Integer | 是 | 被屏蔽的协作伙伴 id（谁的商品被不看） |

---

### 5. 取消「不看他的商品」

**路径：** `POST /unblockPartner`

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| blockerDisId | Integer | 是 | 当前配送商 id |
| blockedDisId | Integer | 是 | 协作伙伴 id |

---

### 6. 不给他看我的商品

**路径：** `POST /setHideMyCatalogFromPartner`

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| myDisId | Integer | 是 | 当前配送商 id |
| partnerDisId | Integer | 是 | 协作伙伴 id |

---

### 7. 取消「不给他看我的商品」

**路径：** `POST /unsetHideMyCatalogFromPartner`

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| myDisId | Integer | 是 | 当前配送商 id |
| partnerDisId | Integer | 是 | 协作伙伴 id |

---

### 8. 订货方获取供货商列表（简要）

**路径：** `GET /disGetOfferDis/{id}`

**路径参数：** `id` = 订货方配送商 id

**返回：** 协作伙伴列表，每个伙伴带 `itemData.blockedByMe`、`itemData.blockedMe`

---

## 三、页面设计建议

### 1. 协作伙伴列表页

- 展示伙伴列表，含账单统计
- 每个伙伴可操作：
  - 开关「不看他的商品」
  - 开关「不给他看我的商品」
  - 解除协作

### 2. 建立协作页

- 选择对方配送商
- 选择邀请类型：**邀请成为我的供货商** / **邀请购买我的产品**
- 提交后调用 `saveBusiness`

### 3. 协作伙伴详情 / 设置

- 展示双向可见性状态
- 两个开关：不看他的 / 不给他看我的

---
